# TC10 — Thaumcraft 4 + 6 · 最终融合

#这是什么？

**这不是 TC7。这不是 TC4 重制也这不是 TC6 续作。**

这是 **TC4 的研究系统**，**TC6 的灵气场系统**，**玩家社区的无限创意**，在 NeoForge 上的**最终融合**。

> *“TC4 的研究桌，TC6 的灵气场，Azanor 的遗产。”*

---

##  核心目标

-  **保留 TC4 的精髓** —— 研究桌、节点、邪术、傀儡、经典坩埚
-  **保留 TC6 的精髓** —— 灵气场、咒波机制、升级版注魔
-  **新增“神秘热力学派”** —— 源质物流、咒波回收、机械动力联动
-  **终局内容** —— 元始节点、终极名牌、名人堂双榜机制
-  **致敬 Azanor** —— 他的名字永远在手册第一页

---

## 社区投票机制

TC10 的新功能**不是淘汰制**，是**排队制**。

1. 不定期在 Issue 区提出新功能
2. 社区投票（每人 3 票）
3. 得票最高的功能进入下一版开发计划
4. 落选功能不会删除，排到队列后面

> **——没有赢家通吃，只有先来后到。**
